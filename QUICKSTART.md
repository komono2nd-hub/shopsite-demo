# 🚀 クイックスタートガイド

マチカド！デモ版を5分で起動する方法

---

## 📥 ダウンロード

### 方法1: ZIPをダウンロード（簡単）

1. GitHubリポジトリページの「Code」ボタンをクリック
2. 「Download ZIP」を選択
3. ダウンロードしたZIPを解凍

### 方法2: Gitでクローン

```bash
git clone https://github.com/YOUR_USERNAME/machikado-demo.git
cd machikado-demo
```

---

## 💻 起動方法

### 🌟 最も簡単な方法

**ブラウザで直接開く**

```bash
# Macの場合
open index.html

# Windowsの場合
start index.html

# Linuxの場合
xdg-open index.html
```

### 🔥 推奨方法（ローカルサーバー）

**Python 3がある場合**

```bash
python3 -m http.server 8000
```

ブラウザで開く: `http://localhost:8000`

**Node.jsがある場合**

```bash
npx http-server .
```

**VS Codeを使う場合**

1. VS Codeでフォルダを開く
2. 拡張機能「Live Server」をインストール
3. `index.html` を右クリック→「Open with Live Server」

## 🎮 操作方法

### 1. トップページ

- 「さっそく探す」ボタンで商品一覧へ
- 人気商品カードをクリックで商品詳細（未実装）
- 出品者カードをクリックで出品者ページ（未実装）

### 2. 商品一覧

- 商品カードの「カートへ」でカートに追加
- 右上のカートアイコンに数字が表示される

### 3. カート

- 「+」「−」ボタンで数量変更
- 「削除」ボタンで商品削除
- 「購入手続きへ」は未実装（トーストが表示されます）

### 4. モバイル表示

- ブラウザの開発者ツール（F12）を開く
- デバイスツールバーをON（Ctrl+Shift+M）
- iPhone/Androidモードで確認

## 🐛 よくある問題

### 画像が表示されない

→ インターネット接続を確認してください（Unsplash APIを使用）

### カートが動作しない

→ JavaScriptが有効か確認してください
→ ブラウザのコンソール（F12）でエラーを確認

### レイアウトが崩れる

→ 最新版のブラウザを使用してください
→ キャッシュをクリアしてください

## 🎨 カスタマイズ

### 色を変更

`css/common.css` の最初の部分:

```css
:root {
    --color-primary: #D4622A;  /* ← この色を変更 */
}
```

### 商品を追加

1. `index.html` や `products.html` の商品カード部分をコピー
2. 画像URLと商品名を変更
3. 商品IDを新しい番号に変更

### ロゴを変更

すべてのHTMLファイルの:

```html
<h1>マチカド！</h1>
```

を変更
