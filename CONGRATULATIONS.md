## 💡 技術的なハイライト

### 1. 軽量な認証システム

```javascript
// たった1行でログイン
auth.login('buyer@example.com', 'password');
```

### 2. 柔軟な権限管理

```javascript
// 購入者＋出品者の両方の権限にも対応
auth.hasRole('buyer');  // true
auth.hasRole('seller'); // true (bothの場合)
```

### 3. 自動ヘッダー更新

```javascript
// ログイン状態に応じて自動更新
auth.updateHeader();
```

### Q: ログインできない

```
A: デモアカウントを使用してください
   buyer@example.com / password
```

### Q: カートが消える

```

   ブラウザのシークレットモードでは消えます
```
