```bash

```

### ステップ2: 2〜3分待つ

GitHub Pagesのビルドを待ちます。

### ステップ3: テスト

```
1. https://komono2nd-hub.github.io/shopsite-demo/login.html を開く

2. F12 を押して開発者ツールを開く

3. Console タブを開く

4. ログインフォームに入力:
   Email: buyer@example.com
   Password: password

5. 「ログイン」ボタンをクリック

6. コンソールに以下のようなログが表示されるはず:
   🔍 login.html スクリプト読み込み完了
   auth オブジェクト: object
   ✅ ログインフォームが見つかりました
   🔐 handleLogin 関数が呼ばれました
   入力値 - Email: buyer@example.com Password: 入力あり
   ログイン結果: {success: true, user: {...}}
   ✅ ログイン成功!
```
